# Projeto desenvolvido no curso de programação da alura

 - Jogo de adivinhação de um número secreto. O HTML e CSS já viram prontos para nós alunos e durante o curso foi praticado a lógica com o JavaScript, trabalhando condicionais, blocos de repetição, variáveis, métodos, funções, arrays e exibição de mensagens para o usuário.
